# 🌦️ Weather App  — Python GUI Project

> A modern, feature-rich weather application built using Python.  
> This app provides real-time weather updates with a clean GUI, gradient background, emoji-based weather indicators, and live map integration.

---

## 🚀 Features

- 🌡️ Real-time weather data using OpenWeather API  
- 📍 Auto location detection (IP-based)  
- 🔍 Search weather by city name  
- 🌈 Beautiful gradient background (Cyan → Light Pink)  
- 😀 Emoji-based weather display (no boring icons)  
- 🌍 Interactive map showing exact location  
- 🖥️ Clean and modern GUI using Tkinter  

---

## 🧠 Concepts Used

- API Integration  
- JSON Data Handling  
- GUI Development (Tkinter)  
- Map Integration (TkinterMapView)  
- Exception Handling  
- Dynamic UI Updates

## ⚙️ Installation

### 1. Clone the Repository
```
git clone https://github.com/yourusername/weather-app.git
cd weather-app-pro
```

### 2. Install Dependencies
```
pip install requests tkintermapview
```

---

## 🔑 API Key Setup

1. Go to: https://openweathermap.org/api  
2. Create an account  
3. Generate your API key  
4. Replace in code:
```
api_key = "YOUR_API_KEY"
```

---

## ▶️ How to Run

```
python weather_app.py
```
